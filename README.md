# UART
## UART Controller with FIFO
### UART interface for PC communication, including transmit and receive logic with FIFO buffering to handle asynchronous data smoothly.
- Control via 16 pins on the bottom side of the board for sending data to PC, displaying received data received from PC on two seven-segment (HEX number symbol of ASCII) displays on the Nexys A7-50T board.
- Testing communication through the PUTTY interface with a computer.
- Confirming data transmission from the board using buttons.
- Confirming data transmission from the PC using keyboard.
- Configuration: parity, 1/2 stop bits, number of data bits 5–9, baud rate 9600/19200.
